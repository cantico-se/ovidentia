OVIDENTIA FX


OVIDENTIA : more than content management... a real tool for collaborative work!

OVIDENTIA is a toolbox allowing to publish with extreme ease and very fast an intranet, extranet or internet portal. Starting with the classic functions of a Content Management System (CMS) such as:

  • publishing of information (WYSIWYG editor, article tree, categorization),
  • validation workflow (allowing to define approbation schemes, from simple ones to the most complex),
  • a search engine,
  • ...


... OVIDENTIA also has powerful collaborative functions

  • Profile manager, shared agendas, mail interface, directories,
  • file manager (with version management)
  • Forums,
  • FAQ,
  • Management of vacation requests (with workflow)
  • Possibility to delegate administration functions (within a given perimeter and for a defined set of functions)
  • ...

Its architecture, completely modular, allows to plug in additional extension modules developed by the OVIDENTIA community.



For more information : http://www.ovidentia.org
