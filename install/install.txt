

Base Requirements
-----------------
PHP4 (http://www.php.net)
MySQL database server (http://www.mysql.com)

Unzip the package into your web server root.

- Create a database called, for example, ovidentia:

    mysqladmin create ovidentia
    
- Fill database with the tables in babinstall.sql file:

    mysql ovidentia < babinstall.sql

- copy  config.php and index.php files located in 'ovidentia31/example' directory 
  to you web root folder and change those variables:
	
	$babDBHost		Mysql Host name
	$babDBLogin		Mysql Login 
	$babDBPasswd	Mysql Password
	$babDBName		Mysql database name ( for example ovidentia )
	
- Now pointe your brower to http://your-site/

- That's all
