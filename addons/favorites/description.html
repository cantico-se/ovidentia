Description

This add-on is mainly a tutorial for the development of an Ovidentia add-on. The add-on enables registerd users to create and maintain a list of "favorites" or "bookmarks" for frequently visited web sites.

When a user clicks the "favorites" menu item in the "User's section", a page is displayed with his list of favorites. Aside each favorite is a link allowing to delete this favorite.

The description of the favorite itself is a link, pointing to the bookmarked site.

An "Add" menu item in the main menu bar displays a new page allowing the creation of a new favorite to bookmark.

On user deletion, all entries for this user in the favorites_list table are deleted.

Database modifications.

At installation time of the add-on a database table "favorites_list" will be created in the Ovidentia database. This table will hold "favorites" entries. Each entry has:

  1. id: auto-increment primary key
  2. id_owner: the user id as given by Ovidentia
  3. url: the url of the favorite to bookmark
  4. description: a text description of this favorite
