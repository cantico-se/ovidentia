Installation:

1- Create database tables in Ovidentia database by executing the dbupdate.sql command file with the mysql client.

2- Create a folder 'workrequest' under $babInstallPath/addons
and copy files from 'programs' folder to this newly created folder.

3- Create a folder 'workrequest' under $babInstallPath/skins/ovidentia/templates/addons
and copy files from 'skins/ovidentia/templates' folder to this newly created folder.

4- Create a folder 'workrequest' under $babInstallPath/lang/addons
and copy files from langfiles folder to this newly created folder.

5- Login as Administrator and give access right to this add-on by using
'addons' menu from Administrator section.

That's all.
